Program Description

This is a personal home automation messaging program operated by James Morse. The program sends SMS notifications and receives SMS commands for controlling a residential garage door. Messages are sent exclusively to and from the system owner's personal mobile number.

Message Frequency

Message frequency varies based on usage. Typical usage is fewer than 20 messages per day. Messages are triggered by vehicle proximity to the owner's residence or by direct SMS commands from the owner.

Message and Data Rates

Message and data rates may apply based on your mobile carrier plan.

Opt-Out

To stop receiving messages, reply STOP to any message. This will disable the automation system's SMS functionality.
